# snyk-threadfix

## Requirements
+ python3
+ pipenv

## Steps to Run
```
git clone https://github.com/snyk-labs/snyk-threadfix.git
cd snyk-threadfix
pipenv install
pipenv run python main.py --orgId=<org_id> --repoName=<repo_name>
```

Additional input parameters are available:
```
pipenv run python main.py --help
```
